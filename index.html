Caveats

My first assignment from my summer 2018 CPSC490 class was to build a website utilizing HTML5, CSS3 and Javascript. This website serves as a short experiment to pull together as much HTML/CSS/JS knowledge as I can in a short period of time (around 13 hours was spent to learn and apply the knowledge) and as a repository for some of my travel images.

Thus, I consider this an incomplete website, with many areas that necessitate tweaking and fixing. For starters, this website is optimized for the newest version of Chrome. If you are using another browser, it is highly likely that all horizontal galleries are underlined by a rather unsightly scrollbar. Or, the horizontal galleries may not scroll at all. Given that this was a short experiment, please do not consider this reflective of my design principles, or writing skills!

Memorable learnings from this project

  • Using JS to manipulate the DOM took up the bulk of my time. Unlike HTML and CSS, my experience with JS was much more limited.
  • In particular, I tried to separate a .js function into two, in line with cohesion principles. However, this would involve calling a dynamically created div by ID which was a problem as the DOM was not constantly refreshed.
  • Finding a way to load a .json file locally
  • Styling the mailto link; it does not count as "visited" so a:visited does not work
  • I got to practice GitHub via the command line and it was my first time setting up a Github page

Known issues and future improvements

  • Clean up the JS code (see above) so that it adheres better to the principle of coupling
  • Find a way for horizontal scrolling to be as unobtrusive across different browsers
  • Find a better font
  • Fix the vertical alignment of the caption overlay
  • Create a responsive version of the website?
